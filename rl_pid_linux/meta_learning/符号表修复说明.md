# 符号表修复说明

## 问题
1. Acronyms表格和Mathematical Symbols表格右边没有对齐
2. Mathematical Symbols表格超出边界，跑到右边栏

## 解决方案

### 1. 统一使用 `tabular*{\linewidth}`

**修改前**:
```latex
\begin{tabular}{@{}ll@{}}  % 缩略词表
\begin{tabular}{@{}lll@{}} % 符号表
```

**修改后**:
```latex
\begin{tabular*}{\linewidth}{@{}l@{\extracolsep{\fill}}l@{}}  % 缩略词表
\begin{tabular*}{\linewidth}{@{}l@{\extracolsep{\fill}}ll@{}} % 符号表
```

### 2. 关键技术点

#### `\linewidth`
- 自动适配当前列宽度
- 在单列模式下约为整页宽度
- 确保两个表格宽度完全一致

#### `@{\extracolsep{\fill}}`
- 在列之间插入可伸缩的空白
- 让表格自动扩展到设定的宽度
- 实现列内容的合理分布

#### 列定义解析

**缩略词表**: `@{}l@{\extracolsep{\fill}}l@{}`
- `@{}` - 移除左边界空白
- `l` - 第一列（Acronym）左对齐
- `@{\extracolsep{\fill}}` - 自动填充空白
- `l` - 第二列（Description）左对齐
- `@{}` - 移除右边界空白

**符号表**: `@{}l@{\extracolsep{\fill}}ll@{}`
- `@{}` - 移除左边界空白
- `l` - 第一列（Symbol）左对齐
- `@{\extracolsep{\fill}}` - 自动填充空白
- `l` - 第二列（Description）左对齐
- `l` - 第三列（Dimension/Value）左对齐
- （隐式）`@{}` - 移除右边界空白

### 3. 简化描述文字

为避免文字过长导致换行，简化了部分描述：
- "Sigmoid activation function" → "Sigmoid activation"
